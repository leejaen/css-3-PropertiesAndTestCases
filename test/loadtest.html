General Browser Load-Time Test

This page tests loading time using several timers - First Access is measured by the first time the DOM can access an element at the top of the page (timer is called First Access); Doc Load triggers when the browser tells javascript the document is loaded (but images etc are still being downloaded); Doc+Images Load is a standard onLoad document trigger. The page contains linked CSS, linked javascript and 11 images (one linked from the CSS file). Saving this locally and running through a local server is the best way to test rendering speed — remember loading off the web includes substantial latency variation… Remember also that Javascript is not totally reliable to measure performance, Firefox fails to include CSS images in its triggers, and IE simply returns weird values sometimes. Also, just running a test once is really meaningless, you need to interleave and average (minimum of 10 if you want to do statistics!) before anything meaningful is shown. You can also try CSS, table (another version), content vs. presentation and other rendering tests.
