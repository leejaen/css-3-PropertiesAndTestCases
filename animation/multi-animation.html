animation

多动画叠加animation

语法：

-webkit-animation: name duration timing-function delay iteration_count direction;

说明

-webkit-animation-name 这个属性的使用必须结合@-webkit-keyframes一起使用

-webkit-animation-duration 表示动画持续的时间

-webkit-animation-timing-function 表示动画使用的时间曲线 ease | linear | ease-in | ease-out | ease-in-out

-webkit-animation-delay 表示开始动画之前的延时

-webkit-animation-iteration-count 表示动画要重复几次

-webkit-animation-direction 表示动画的方向 normal(默认) | alternate ,normal 方向始终向前,alternate 当重复次数为偶数时方向向前，奇数时方向相反

-webkit-animation-fill-mode : none (默认)| backwards | forwards | both 设置动画开始之前和结束之后的行为

-webkit-animation-play-state: running（默认） | paused 设置动画的运行状态

语法如下：

实例

            <div id="aniDiv" class="ani"></div>
            <div id="aniDiv2"  class="ani"></div>
            <div id="aniDiv3" class="ani"></div>
            <div id="aniDiv4"  class="ani"></div>
            #aniDiv
            {
                -webkit-animation-delay:1.5s;
            }
            #aniDiv2
            {
                
                -webkit-animation-delay:2s;
            }
            #aniDiv3
            {
                border-color:#FFF;
                -webkit-animation-delay:3s;
            }
            #aniDiv4
            {
                border-color:#CF0;
                -webkit-animation-delay:4.8s;
            }
