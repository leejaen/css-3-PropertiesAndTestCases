Selector

E:empty结构性伪类选择符

语法：

E:empty{ sRules }

说明

匹配没有任何子元素（包括text节点）的元素E。

实例

<html>
  <head>
  <style type="text/css">
  li{height:20px; line-height:20px; font-size:12px;}
.a li:nth-last-of-type(6)
  {
  background-color:#0F6;
  }
 </style>
  </head>
  <body >
  <div class="demo">
  <ul class="a">
  <li>a1</li>
  <p>我是p标签</p>
  <li>a2</li>
  <li>a3</li>
  <li>a4</li>
  <li>&nbsp;</li>
  <li><!----></li>
  </ul>
  
  </div>
  </div>
  </body>
  </html>
