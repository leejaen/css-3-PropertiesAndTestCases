Selector

E::selection伪对象选择符

语法：

E::selection{ sRules }

说明

设置对象被选择时的样式。

实例

    <html>
    <head>
    <style type="text/css">
   	 li{height:20px; line-height:20px; font-size:12px; list-style:none; color:#000}
    
    
		.a li::selection{
			color:#F00;
			background-color:#6CC;
		}
		
    </style>
    </head>
    <body>
    <div class="demo">
          <ul class="a">
              <li>我是li标签</li>
          </ul>    
    </div>
	</body>
    </html>

效果预览

  • 我是li标签,选择我试一下
