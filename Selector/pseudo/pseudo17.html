E:after/E::after伪对象选择符

语法：

E:after/E::after{ sRules }

说明

设置在对象后（依据对象树的逻辑结构）发生的内容。用来和content属性一起使用， 默认地，这个伪元素是行内元素，不过可以使用属性 display 改变这一点
CSS3将伪对象选择符(Pseudo-Element Selectors)前面的单个冒号(:)修改为双冒号(::)用以区别伪类选择符(Pseudo-Classes Selectors)，但以前的写法仍然有效。

实例

    <html>
    <head>
    <style type="text/css">
   		 li{height:20px; line-height:20px; font-size:12px; list-style:none; color:#000}
    
    
		.a li::after{
			color:#F00;
			content:"css 添加的内容";
		}
		
    </style>
    </head>
    <body>
    <div class="demo">
          <ul class="a">
              <li>我是li标签</li>
          </ul>    
    </div>
	</body>
    </html>

效果预览
