E:nth-last-child(n)结构性伪类选择符

语法：

E:nth-last-child(n){ sRules }

说明

匹配父元素的倒数第n个子元素E。 要使该属性生效，E对象必须是某个对象的子元素。

实例

 
  <html>
  <head>
  <style type="text/css">
  li{height:20px; line-height:20px; font-size:12px;}	
         .a li:nth-last-child(5)
	{
		background-color:#0F6;
	}
	.b li:nth-last-child(2n+1)
	{
		background-color:#FC3;
	}
   
   
   .b li:nth-child(2n+1)
	{
   		color:red;
    }
</style>
  </head>
  <body >
  <div class="demo">
  <ul class="a">
      <li>a1</li>
      <li>a2</li>
      <li>a3</li>
      <li>a4</li>
      <li>a5</li>
      <li>a6</li>
  </ul>
  <ul class="b">
     <li>b1</li>
     <li>b2</li>
     <li>b3</li>
     <li>b4</li>
  </ul>
  
  </div>
  </div>
  </body>
  </html>
