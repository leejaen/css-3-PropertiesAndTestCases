Selector

E:last-child结构性伪类选择符

语法：

E:last-child{ sRules }

说明

匹配父元素的最后一个子元素E。 要使该属性生效，E对象必须是某个对象的子元素。

实例

  <html>
  <head>
  <style type="text/css">
  li{height:20px; line-height:20px; font-size:12px;}
	.b  li:last-child/*注意用的不是ul:last-child*/	{
		background-color:#FC3;
	}
  </style>
  </head>
  <body >
<div class="demo">
      <ul class="a">
          <li>a1</li>
          <li>a2</li>
          <li>a3</li>
          <li>a4</li>
          <li>a5</li>
          <li>a6</li>
      </ul>
          <ul class="b">
          <li>b1</li>
          <li>b2</li>
          <li>b3</li>
          <li>b4</li>
          <li>b5</li>
          <li>b6</li>
      </ul>
      
  </div>
</div>
</body>
  </html>
